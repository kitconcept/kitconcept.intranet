# kitconcept.intranet

A Plone distribution for Intranets with Plone. Created by kitconcept.

## Installation

Install kitconcept.intranet with `pip`:

```shell
pip install kitconcept.intranet
```
And to create the Plone site:

```shell
make create_site
```
