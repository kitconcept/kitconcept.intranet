# Installation

Install Node 20:

```
nvm install 20
nvm use 20
```

Enable Coremedia:

```
corepack enable pnpm
```

Build:

```
make install
```

# Start

Start Frontend:

```
cd frontend && pnpm start
```

Start Backend

```
make start-backend
```

# Export

Export content to the file system:

```
make export
```
