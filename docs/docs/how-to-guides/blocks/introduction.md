# Use the Introduction block

The Introduction block renders styled introductory text at the top of a page. It functions like a rich-text block but with a restricted toolbar (headings and blockquote are not available) to enforce a consistent visual style.

## Prerequisites

- You have Editor or Manager access to the page.

## Adding the Introduction block

1. Open the page in edit mode.
2. Click the **+** button to open the block chooser.
3. Search for **Introduction** or find it under the **Text** group.
4. Click **Introduction** to insert the block.

## Writing content

Click inside the block and type your introductory text. The toolbar supports bold, italic, links, and inline formatting — heading and blockquote buttons are intentionally removed.

## Enter key behavior

Pressing **Enter** inside an Introduction block inserts a new Introduction block immediately after the current one, instead of creating a default text block. This allows you to build multi-paragraph introductions while keeping each paragraph styled as introduction text.

To switch to a different block type, use the block chooser (**+** button) or type `/` to trigger the inline block search.

## Notes

- The Introduction block is marked as **most used**, so it appears at the top of the block chooser.
- Pressing **Backspace** at the very start of a list inside the block converts the list item back to a plain paragraph.
- The block supports the Slate rich-text editor and manages its own keyboard focus.
