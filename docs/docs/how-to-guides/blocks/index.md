# Working with blocks

These guides explain how to add and configure the blocks available in your kitconcept Intranet.

## Overview

The kitconcept Intranet provides a rich set of content blocks for building pages. In addition to the standard Volto blocks, the following custom blocks are available:

| Block | Purpose | Audience |
|-------|---------|----------|
| [Event Calendar](event-calendar) | Display upcoming events in a calendar or list view | Editors |
| [Contact List](contact-list) | Display a curated list of persons/contacts | Editors |
| [IFrame](iframe) | Embed external web content | Editors |
| [RSS](rss) | Show live RSS feed content | Editors |
| [Slider](slider) | Display content in a slideshow/carousel | Editors |
| [Form](form) | Add interactive forms to pages | Editors |

## Adding a Block

To add any block to a page:

1. Open the page in edit mode.
2. Click the **+** button to add a new block or click on an empty area.
3. Search for the block by name in the block chooser.
4. Click the block to insert it.
5. Configure the block using the sidebar panel on the right.
6. Click **Save** to publish your changes.
